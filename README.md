## Run Locally

Clone the project

```bash
  git clone https://github.com/ssahibsingh/Fashion-Shop
```

Go to the project directory

```bash
  cd React_E-Commerce
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```



## Tech Stack

* [React js](https://reactjs.org/)
